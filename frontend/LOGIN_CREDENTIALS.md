# بيانات تسجيل الدخول

## المستخدمون الافتراضيون

يمكنك استخدام أي من بيانات تسجيل الدخول التالية للدخول إلى المنصة:

### 1. مدير النظام
- **اسم المستخدم:** `admin` أو `مدير`
- **كلمة المرور:** `admin123`

### 2. مستخدم عادي
- **اسم المستخدم:** `user` أو `مستخدم`
- **كلمة المرور:** `user123`

### 3. أحمد
- **اسم المستخدم:** `ahmed` أو `أحمد`
- **كلمة المرور:** `ahmed123`

### 4. فاطمة
- **اسم المستخدم:** `fatima` أو `فاطمة`
- **كلمة المرور:** `fatima123`

### 5. محمد
- **اسم المستخدم:** `mohammed` أو `محمد`
- **كلمة المرور:** `mohammed123`

---

## ملاحظات مهمة

- جميع بيانات تسجيل الدخول حساسة لحالة الأحرف (case-sensitive) في كلمة المرور
- يمكن استخدام الأسماء بالعربية أو الإنجليزية
- هذه بيانات افتراضية للتطوير والاختبار فقط
- في الإنتاج، يجب استبدال هذا النظام بنظام تحقق حقيقي من قاعدة البيانات

---

## كيفية تغيير بيانات تسجيل الدخول

يمكنك تعديل بيانات تسجيل الدخول في الملف:
```
src/pages/login.vue
```

في الكائن `defaultUsers` داخل دالة `handleLogin`.

## الأمان

⚠️ **تحذير:** هذا النظام للاختبار فقط. في الإنتاج يجب:
- تشفير كلمات المرور
- استخدام قاعدة بيانات آمنة
- إضافة نظام جلسات (Sessions)
- إضافة نظام صلاحيات متقدم
- إضافة نظام تسجيل محاولات الدخول
